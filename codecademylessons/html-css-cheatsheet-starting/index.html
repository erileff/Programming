HTML5 New Keywords

Tag Formatted Tag Description
nav<nav>defines a section with navigation links; cannot be nested within another nav (i.e. for secondary menus)
article<article>defines a piece of self-contained content; doesn't refer to main content alone
section<section>defines a section of a document to indicate a related grouping of semantic meaning (i.e. to provide extra context for parent element)
aside<aside>defines a section that is related to the main element but doesn't belong to the main flow, like an explanation box or advertisement
video<video>media player for video; attributes include width and height, autoplay, loop, muted, poster, and preload.
audio<audio>like video, but without a visual component
mark<mark>represents text that's marked or highlighted for reference or notation purposes (i.e. highlighting search term in results)
figure<figure>represents self-contained content, potentially with a caption
figcaption<figcaption>represents a caption or legend describing the rest of the contents of its parent <figure> element.
data<data>links a given piece of content with a machine-readable translation
time<time>represents a specific period in time; may include the datetime attribute to translate dates into machine-readable format, allowing for better search engine results or custom features such as reminders
output<output>a container element into which a site or app can inject the results of a calculation or the outcome of a user action
progress<progress>displays an indicator showing the completion progress of a task, typically displayed as a progress bar
meter<meter>represents either a scalar value within a known range or a fractional value
main<main>represents the dominant content of the <body> of a document
